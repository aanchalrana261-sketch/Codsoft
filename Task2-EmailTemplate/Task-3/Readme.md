# 🍴 Restaurant Menu Page

## 📌 Project Overview
This project is a **Restaurant Menu Page** developed as **Task 3** of the **CodSoft Web Development Internship**. It showcases a visually appealing and responsive menu layout where users can easily browse different food categories and dishes.

## ✨ Features
- Responsive and modern design
- Well-organized food categories
- Attractive menu cards
- Clean and user-friendly interface
- Mobile-friendly layout

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 🎯 Objective
The goal of this project is to design a professional 
restaurant menu page while improving front-end development
skills using HTML, CSS, and JavaScript.



## 🚀 How to Run
1. Clone or download this repository.
2. Open the project folder.
3. Run `index.html` in any web browser.
